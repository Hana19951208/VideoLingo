# Antigravity 行为准则 (Agent Rules)

## 核心交互规范
1. **纯中文交流**：所有的思考、输出文档、代码注释、向用户的回复，都必须严格使用中文。
2. **遵守超级纪律 (using-superpowers)**：在执行任何动作前，强制检查并应用相关的 Skills（如 brainstorming, writing-plans 等），严格按照设计图和清单执行任务。
3. **Markdown 格式优先**：提供结构清晰、重点突出的 Markdown 格式内容，对代码和终端命令提供明确说明。

## 编码与工程规范
1. **Windows 本地支持**：充分结合用户当前提供的 Windows 环境、`docker`（V2 语法）与 `uv`（Python 新一代包管理器）环境实施部署。
2. **保持环境整洁**：所有的操作脚本和命令应该首先考虑可重复性及避免污染全局环境（优先走 `uv venv` 或 Docker 隔离环境）。
3. **架构优化思维**：对于如替换 TTS 模块等微服务抽离调整，应保持代码的 Clean Code 风格，并优先遵循用户给定的 API 契约（如 `API.md`）。

*这份准则是本项目执行阶段的刚性约束，在每次规划与实施时自动生效。*
